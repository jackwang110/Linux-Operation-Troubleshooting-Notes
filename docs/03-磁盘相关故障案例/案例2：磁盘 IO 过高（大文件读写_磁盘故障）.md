# 案例2：磁盘 IO 过高（大文件读写/磁盘故障）

## 现象
- 系统响应缓慢，应用程序卡顿
- `iostat` 显示高 IO 等待时间
- 磁盘读写速度明显下降

## 排查步骤
1. 使用 `iostat -x 1` 查看磁盘 IO 情况
2. 使用 `iotop` 查看进程 IO 使用情况
3. 检查是否有大文件读写操作
4. 检查磁盘健康状态

## 根因分析
- 大文件读写导致磁盘负载过高
- 磁盘故障（坏道、老化等）

## 解决方案
- 优化大文件读写操作
- 更换故障磁盘
- 调整 IO 调度器

## 预防措施
- 定期检查磁盘健康状态
- 优化应用程序 IO 操作
- 考虑使用 SSD 提升性能
