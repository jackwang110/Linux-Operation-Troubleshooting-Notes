## 12. 模板使用说明

### 12.1 填写要求

1. **完整性**：尽量填写所有字段，确保信息完整
2. **准确性**：确保记录的信息真实、准确
3. **详细性**：关键步骤和重要信息要详细记录
4. **及时性**：故障处理完成后及时进行复盘和记录
5. **一致性**：保持格式和术语的一致性

### 12.2 存储和共享

1. **存储位置**：统一存储在知识库中，便于检索和参考
2. **共享范围**：根据故障的敏感程度，确定共享范围
3. **更新维护**：定期更新和维护复盘文档，确保信息的时效性

### 12.3 应用场景

1. **故障处理**：作为故障处理过程的记录工具
2. **知识共享**：作为团队内部知识共享的载体
3. **培训教材**：作为新员工培训和技能提升的教材
4. **流程改进**：作为流程优化和改进的依据
5. **经验传承**：作为组织经验传承的重要方式
