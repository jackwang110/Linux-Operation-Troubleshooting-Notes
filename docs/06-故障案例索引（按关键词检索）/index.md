## 使用指南

1. **关键词检索**：在"关键词索引"中查找与问题相关的关键词，点击对应案例查看详细内容。

2. **故障类型检索**：根据故障的类型（如系统资源故障、网络故障、服务故障）在"按故障类型索引"中查找相关案例。

3. **排查工具检索**：根据使用的排查工具在"按排查工具索引"中查找相关案例，了解工具的使用场景和方法。

4. **解决方案检索**：根据可能的解决方案在"按解决方案索引"中查找相关案例，了解解决方案的适用场景和实施方法。

5. **交叉检索**：结合多个索引维度进行交叉检索，提高检索的准确性和效率。
